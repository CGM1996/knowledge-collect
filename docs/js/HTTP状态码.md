
### 状态总结
| 状态码  | 含义       |
| ------- | ---------- |
| 100-199 | 信息响应   |
| 200-299 | 成功响应   |
| 300-399 | 重定向     |
| 400-499 | 客户端错误 |
| 500-599 | 服务器错误 |


  - 常见http状态码   （302    location  跳转）
