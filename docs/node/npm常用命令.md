## npm 常用命令

* npm root -g 查找node包安装的全局目录
* npm root 查找本项目下node包安装的目录
* npm config set register XXX
* npm config list
* npm config ls
### 用户
* npm adduser
* npm whoami

### npm源
* npm config get registry
* npm config set registry XXX

### nrm
```
npm i nrm -g
```
* nrm ls // 查看所有的npm 源
* nrm use taobao  // 切换源
